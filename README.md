## 3. 아키텍처 구조

```
[Browser] → JSP
    ↕ Ajax / Form Submit
[Controller] (Spring MVC)
    ↕ Service Layer
[Service / ServiceImpl]
    ↕ DAO Layer
[DataAccessObject] + MyBatis XML Mapper
    ↕
[MySQL DB]
```

### 패키지 구조 (net.su.*)

계층형 아키텍처(Controller → Service → DAO → VO) 기반으로 아래 도메인 영역으로 구성:

| 패키지 | 담당 도메인 |
|--------|------------|
| `login` | 로그인, 회원관리, 게시판, 암호화 |
| `customer` | 고객, 가족구성원, 목표, 설계자 관리 |
| `asset` | 자산 전체 CRUD (유동성·현금성·채권·주식·부동산·대출·보험·카드 8종) |
| `cusPage` | 고객 전용 포트폴리오 조회 페이지 |
| `output` | 투자 성과 분석 및 출력 화면 |

## 4. 주요 기능

### 자산 관리
- 8가지 자산 유형(유동성·현금성·채권형·주식형·부동산·대출·보험·신용카드) CRUD
- 자산별 계약·납입·만기·수익자 정보 통합 관리
- 벤치마크 지수 등록 및 관리 (국내외 주요 지수)

### 투자 성과 분석
- **FCR (미래현금소요)**: 고객 인생 목표별 필요 자금과 현재 적립 상황 비교 분석
- **IRR (내부수익률)**: 투자자산별 현금흐름 기반 실제 수익률 계산 및 벤치마크 비교
- **투자매트릭스**: 자산유형 × 투자등급 기반 포트폴리오 구성 시각화
- **TAA (전술적 자산배분)**: 현재 자산배분 vs 목표 배분 비교 및 리밸런싱 제안

### 지출·현금흐름 분석
- 지출 카테고리별 집계 및 월별 현금흐름 분석
- 연간 지출·정기비용 관리
- 카드지급 연동 추적

### 기타
- 고객 전용 포트폴리오 조회 페이지 (별도 접근 권한)
- 집계 데이터 엑셀 출력 지원
- 시계열 재무지표 추세 분석

## 5. 메뉴 구조

```
[로그인]
    ↓
[홈 대시보드]
    │
    ├── 고객 관리
    │   ├── 고객 목록 / 등록 / 수정 / 삭제
    │   ├── 가족구성원 관리
    │   ├── 목표 설정 (16가지 인생목표 + 고객별 목표금액/시점)
    │   └── 담당 설계자 정보
    │
    ├── 자산 관리
    │   ├── 자산 목록 (전체 자산 현황)
    │   ├── 유동성 자산 (입출금 통장 — 잔액, 마이너스대출)
    │   ├── 현금성 자산 (예금/적금 — 금리, 만기금액)
    │   ├── 채권형 (채권/채권형펀드 — 투자상세 매수/매도)
    │   ├── 주식형 (주식/주식형펀드 — 투자상세 매수/매도)
    │   ├── 부동산 / 기타자산
    │   ├── 대출 (대출금액, 금리, 상환방식)
    │   ├── 보험 (납입, 만기수령, 피보험자)
    │   ├── 신용카드
    │   └── 벤치마크 등록 / 수정 (지수별 데이터 관리)
    │
    ├── 투자현황 분석
    │   ├── 통장현황 — 전체 입출금 통장 잔액 현황
    │   ├── 지출현황 — 지출 계정별 현황 요약
    │   ├── 지출지급 — 지출 항목별 지급 내역
    │   ├── 세금체크 — 절세/세금 계산
    │   └── 현금흐름체크 — 월별 현금흐름 분석
    │
    ├── 투자결과
    │   ├── 지속투자결과 — 현재 전략 유지 시 미래 결과 분석
    │   └── 자체투자결과 — 고객 자체 운용 실적
    │
    ├── FCR 분석
    │   └── 미래현금소요 — 목표 달성을 위해 향후 필요한 현금 소요량 계산 및 부족분 시각화
    │
    ├── 투자매트릭스
    │   └── 자산유형 × 투자등급 매트릭스 분석 (채권형/주식형 포트폴리오 구성 시각화)
    │
    ├── TAA (전술적 자산배분)
    │   └── 현재 자산배분 vs 목표 자산배분 비교 및 리밸런싱 제안
    │
    ├── 투자상세 테이블
    │   └── 자산별 매수/매도 전체 내역 통합 조회
    │
    ├── 보조데이터
    │   ├── 투자현황표 — 전체 자산 현황 요약표
    │   ├── 집계시트 — 통합 집계 (엑셀 출력 지원)
    │   ├── 카드지급 — 신용카드 지급 내역
    │   ├── 날짜관리 — 기준일 관리 (엑셀 출력 지원)
    │   ├── 재무추세 — 시계열 재무지표 추세
    │   └── 입출금추적 — 상품별 입출금 추적
    │
    ├── 고객 전용 페이지
    │   └── 고객이 직접 본인 포트폴리오를 조회하는 화면
    │
    └── 게시판 / 공지사항
```

## 7. 개발 이력별 주요 변경사항


### 1차 개발 (2015)
- 핵심 자산관리 DB 설계 및 기본 CRUD 구현
- 계약서 기준 기능명세서 작성 및 ERD 설계
- Spring MVC + MyBatis 기반 프로젝트 구조 수립

### 2차 개발 (2016)
- 분석·출력 기능 대거 추가 (FCR, TAA, 투자매트릭스, 투자결과, 현금흐름 등)
- 전체 Java 소스 약 200여 개 파일로 확장
- Bootstrap 기반 반응형 UI 완성

### 3차 추가개발 및 유지보수 (2017)
- 재무전문가 소개 홈페이지 추가
- 고객 계좌·메모 관리 기능 신규 추가
- PMT 계산 기능 (목표금액 ↔ 월납입금 자동 계산)
- 상품별 투자 Tracking 기능 추가
- 시점별 데이터 스냅샷 저장 기능 추가
